About This App
MyTodo is a simple Flask-based ToDo list app that lets users create, view, update, and delete tasks. It uses SQLite for database storage and Bootstrap for styling.
Help
• Use the form to add a todo.
• Use the “Update” and “Delete” buttons to manage tasks.
• View all todos under the table.
• “Completed” and “Pending” filters are coming soon!
